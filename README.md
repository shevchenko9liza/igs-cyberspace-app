## Проблема

Страхование -- одна из самых непонятных финансовых тем для подростков. Термины вроде "франшиза", "страховая премия", "полис" звучат как другой язык. Страховые компании традиционно общаются через родителей -- или не общаются вообще.

**Результат:** поколение, которое не понимает зачем нужна страховка, не умеет оценивать риски и впервые сталкивается с этими понятиями уже во взрослой жизни -- часто в стрессовой ситуации.

## Решение

**CyberSpace & Mentor** -- интерактивный сценарий, в котором подросток **на практике** понимает как работает страхование. Не лекция. Не видео. Не тест. А **опыт**: ты зарабатываешь, покупаешь вещи, сталкиваешься с рисками -- и видишь, как страховка экономит деньги.

| Игровое действие | Реальный урок |
|---|---|
| Купить страховку на смартфон | Зачем нужна имущественная страховка |
| Страховой случай с франшизой 20% | Что такое франшиза (твоя часть оплаты) |
| Скам на скины CS:GO -- страховка не работает | Мошенничество не покрывается страховкой |
| Взять кредит в банке | Как работает кредитный лимит и проценты |
| YouTube-канал генерирует монеты | Что такое пассивный доход |
| Пени за просроченный долг | Почему важно платить вовремя |

## Ключевые фичи

### Пошаговый tutorial (8 шагов)
- Собака-ментор ведёт за руку: заработай -> застрахуй -> столкнись с инцидентом
- Подсветка нужного таба, автопереход на информационных шагах
- Форсированный инцидент в нужный момент для демонстрации страховки
- Кнопка "Пропустить" для опытных пользователей

### Квиз проверки знаний (5 вопросов)
- Появляется после tutorial автоматически
- Вопросы с 4 вариантами ответа и пояснениями
- Темы: франшиза, выгода страховки, нестрахуемые случаи, расчёт риска
- Результат сохраняется -- доступен в табе "Ментор"

### AI-Ментор на Claude Sonnet 4.6
- Встроенная база знаний о страховании РФ (ОСАГО, КАСКО, ОМС, ДМС)
- Информация об Ингосстрахе и рынке страхования
- Off-topic фильтр: вопросы не про финансы вежливо отклоняются
- Fallback-ответы при недоступности API
- Temperature 0.3 + maxTokens 300 = минимум галлюцинаций
- Rate limiting: 15 запросов/мин

### Система страхования
- Одноразовая страховка на каждый актив (сгорает после использования)
- Франшиза 20% -- оплачиваешь часть ущерба сам
- 3 кнопки при ЧП: страховая выплата / из кармана / спросить ментора
- Предупреждение если монет не хватает (разница уйдёт в долг)
- Нестрахуемые случаи (скам) -- учит что мошенничество не покрывается

### Живой игровой мир (Phaser 3)
- Изометрический 2D-рендеринг на движке Phaser 3
- 3 локации: Стартовая комната -> Квартира -> Частный дом
- Предметы появляются с Bounce-анимацией при покупке
- Пёс-ментор с анимацией дыхания
- Дымовые частицы над сломанными предметами
- Плавающие "+20" при заработке

### Банковская система
- Кредитный лимит 5000 монет с учётом 15% комиссии
- Пеня +5% каждые 60 секунд при просрочке
- Автоматическое предупреждение о долговых последствиях

## Бизнес-логика

### Для кого этот продукт

**Прямой заказчик:** страховая компания (Ингосстрах или любая другая)
**Конечный пользователь:** подростки 11-18 лет

### Зачем страховой компании подростки?

Подростки не купят полис сегодня. Но бренд, который **первым объяснил** им страхование понятным языком, получает преимущество на годы вперёд.

```
  СЕГОДНЯ                              ЧЕРЕЗ 3-5 ЛЕТ
     |                                      |
  Подросток играет                    Молодой взрослый
  в CyberSpace                        выбирает первую страховку
     |                                      |
  Узнаёт: франшиза,                  Помнит бренд, который
  ОСАГО, риск                         объяснил первым
     |                                      |
  Brand Consideration                 Конверсия в клиента
  формируется                          с минимальным CAC
```

### Модель монетизации (B2B)

| Канал | Описание |
|---|---|
| **White-label** | Брендированная версия для сайта/приложения страховой |
| **Образование** | Интеграция в школьные курсы финансовой грамотности |
| **Lead generation** | После сценария -- предложение реального продукта |

### Метрики пилота

| Метрика | Что измеряет | Реализация |
|---|---|---|
| **Scenario Completion Rate** | Дошёл ли до конца tutorial | Флаг `tutorialCompleted` |
| **Knowledge Lift** | Рост понимания страхования | Квиз 5 вопросов, score |
| **Insurance Action Rate** | Купил/использовал страховку | Трекинг в игре |
| **Useful Session Rate** | Полезна ли сессия | Composite метрика |
| **Brand Consideration Lift** | Отношение к страхованию | Post-survey (этап 2) |

## Технический стек

| Слой | Технология | Зачем |
|---|---|---|
| **Frontend** | Next.js 15.5, React 19, TypeScript | SSR, быстрый деплой, типобезопасность |
| **Игровой движок** | Phaser 3.85 | Изометрическая 2D-комната, анимации, VFX |
| **Стейт** | Zustand 5 + persist | Реактивное состояние + localStorage |
| **AI-ментор** | Claude Sonnet 4.6 (Anthropic) | Минимум галлюцинаций |
| **UI** | Tailwind + Shadcn/ui + Radix | Тёмная тема, доступность |
| **Деплой** | Vercel | Автодеплой из GitHub |

### Архитектура

```
cyberspace-app/
├── src/
│   ├── app/

│   │   ├── layout.tsx               # Root layout, viewport meta, шрифты
│   │   └── api/chat/route.ts        # AI Mentor: Claude + guardrails + fallback

│   │   └── MainScene.ts             # Phaser 3: изометрия, анимации, частицы
│   ├── store/

│   ├── constants/
│   │   ├── gameData.ts              # Предметы, локации, инциденты, декор
│   │   ├── tutorialSteps.ts         # 8 шагов обучения
│   │   └── quizData.ts              # 5 вопросов квиза
│   └── components/
│       ├── PhaserGame.tsx            # Обёртка Phaser с динамическим импортом
│       └── ui/                       # Shadcn/ui компоненты
├── public/                           # Игровые ассеты (фоны, спрайты, маскот)
└── docs/                             # Скриншоты для README
```

### Защита от галлюцинаций LLM (5 уровней)

| Уровень | Механизм |
|---|---|
| 1 | Встроенная база знаний (ОСАГО, КАСКО, ОМС, ДМС, Ингосстрах) в system prompt |
| 2 | Off-topic фильтр -- отклоняет вопросы не про финансы без вызова API |
| 3 | Fallback-ответы из проверенного словаря при недоступности API |
| 4 | Temperature 0.3 + maxTokens 300 -- минимальная вариативность |
| 5 | Rate limiting 15 req/min на IP -- защита от спама |

## Запуск

### Онлайн (рекомендуется)

**[igs-cyberspace-app-owjc.vercel.app](https://igs-cyberspace-app-owjc.vercel.app)**

### Локально

```bash
git clone https://github.com/shevchenko9liza/igs-cyberspace-app.git
cd igs-cyberspace-app
npm install --legacy-peer-deps
```

Создайте `.env.local`:
```env
ANTHROPIC_API_KEY=your_key_here
```

```bash
npm run dev
# -> http://localhost:3000
```
